# Blender_Learn

단축키
---
ctrl + shift + 마우스 휠클릭 - > 느리게 줌인, 아웃
shift + 마우스 휠클릭 -> 옆에서 보기
숫자패드 . -> 물체를 바로 확대
shift + 스페이스바 -> 기즈모 메뉴
G -> 잡기
tap -> 오브젝트 모드 - 에디트 모드 변환
왼쪽위 메뉴 3개 -> 버텍스, 에지, 페이스 (꼭짓점, 모서리, 면)
shift + 왼쪽클릭 -> 다중 선택
S + X, Y, Z -> x y z축으로 스케일 조정
R + X, Y, Z -> x y z축으로 회전 (숫자로 값을 줄 수도 있음
